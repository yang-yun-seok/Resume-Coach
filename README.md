# 자소서 코치 (Resume Coach)

**게임업계 취업 준비생을 위한 자기소개서 및 경험 분석 워크스페이스**

![자소서 코치](https://i.postimg.cc/w13sbxzS/image.png)

"자소서 코치"는 게임업계(기획, 프로그래밍, 아트, QA, GM 등) 취업을 준비하는 지원자들이 자신의 경험과 이력서 초안을 입력하면, **실무 면접관의 시선**에서 분석하여 논리적인 피드백과 실무 용어로 다듬어진 결과물을 제공하는 AI 기반 웹 어플리케이션입니다.

## ✨ 주요 기능

*   **직무 맞춤형 분석**: 지원하는 세부 직무(예: 유니티 클라이언트, 레벨 디자인, 3D 아트, QA 등)에 맞춰 완전히 다른 관점의 실무 피드백을 제공합니다.
*   **다양한 페르소나 코칭**: '따뜻한 선배(착한맛)'부터 '극대노 면접관(불닭맛)'까지 코칭의 강도를 조절하여 원하는 스타일의 피드백을 받을 수 있습니다.
*   **STAR / PREP 구조화**: 작성된 경험을 자기소개서에 적합한 **STAR 기법**(Situation, Task, Action, Result)과 1분 면접 답변에 적합한 **PREP 기법**(Point, Reason, Example, Point)으로 자동 재구성해 줍니다.
*   **PDF 파싱 지원**: 이력서나 포트폴리오 PDF 파일을 드래그 앤 드롭하면 자동으로 텍스트를 추출해 분석합니다.
*   **보안 철저 (BYOK 방식)**: 입력한 Gemini API 키는 오직 브라우저의 로컬 스토리지에만 저장되며 외부 서버로 절대 전송되지 않습니다.

## 🛠️ 기술 스택 (Tech Stack)

*   **프론트엔드**: HTML5, CSS3 (Vanilla), JavaScript (ES6+)
*   **AI 모델**: Google Gemini API (2.5 Flash / Pro 등 지원)
*   **라이브러리**: PDF.js (PDF 텍스트 추출용)

## 🚀 시작하기 (Getting Started)

이 프로젝트는 백엔드 서버 없이 클라이언트 사이드에서만 동작하도록 설계되었습니다.

1.  이 저장소를 클론(Clone)하거나 다운로드합니다.
2.  폴더 안의 `index.html` 파일을 더블클릭하여 웹 브라우저에서 엽니다. (별도의 로컬 서버 구축 없이도 완벽하게 동작합니다.)
3.  우측 상단의 **[설정]** 버튼을 클릭하여 본인의 **Gemini API Key**를 입력하고 저장합니다.
    *   API 키는 [Google AI Studio](https://aistudio.google.com/app/apikey)에서 1분 만에 무료로 발급받을 수 있습니다.
4.  지원할 직무를 선택하고, 분석할 경험이나 자소서 초안을 입력한 뒤 **[AI 코칭 시작]**을 클릭하세요!
